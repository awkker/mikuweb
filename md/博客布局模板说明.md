# 博客布局模板说明

## 概述

创建了一个博客风格的两栏布局模板，用于展示画师作品集。左侧为文章/作品列表，右侧为作者信息卡片。

## 文件结构

```
static/
├── css/
│   └── blog-layout.css    # 博客布局样式
└── html/
    └── china/
        └── Mitcha.html    # 抹茶老师作品集页面
```

## 布局说明

### 整体结构

```
┌─────────────────────────────────────────────────────┐
│                     导航栏                           │
├──────────────────────────────┬──────────────────────┤
│                              │      作者头像         │
│    作品卡片 1                │      作者名称         │
│    [封面图] + [标题/简介]    │      作者简介         │
│                              │   ───────────────    │
├──────────────────────────────┤   作品 | 分类 | 标签  │
│                              │    24  |  4   | 12   │
│    作品卡片 2                │   ───────────────    │
│    [封面图] + [标题/简介]    │   🎨  🐦  ✉️  🏠      │
│                              │      社交链接         │
├──────────────────────────────┴──────────────────────┤
│                      ...                             │
└─────────────────────────────────────────────────────┘
```

### 左侧：文章卡片

```html
<div class="article-card" data-id="1">
    <div class="article-info">
        <span class="article-category">分类名称</span>
        <h2 class="article-title">作品标题</h2>
        <p class="article-excerpt">作品简介描述...</p>
        <span class="article-date">2025-4-7</span>
    </div>
    <div class="article-cover">
        <img src="图片路径" alt="封面">
    </div>
</div>
```

### 右侧：作者卡片

```html
<div class="author-card">
    <div class="author-avatar">
        <img src="头像路径" alt="作者名">
    </div>
    <h3 class="author-name">作者名称</h3>
    <p class="author-bio">作者简介</p>
    <div class="author-stats">
        <div class="stat-item">
            <span class="stat-label">作品</span>
            <span class="stat-value">24</span>
        </div>
        <!-- 更多统计项... -->
    </div>
    <div class="author-links">
        <a href="#" title="Pixiv">🎨</a>
        <!-- 更多链接... -->
    </div>
</div>
```

## 功能特性

### 交互效果

| 交互 | 效果 |
|------|------|
| 卡片悬停 | 上浮 + 阴影增强 + 图片缩放 |
| 点击卡片 | 弹出详情弹窗 |
| ESC / 点击背景 | 关闭弹窗 |
| 作者卡片 | 右侧固定跟随滚动 |

### 响应式设计

| 屏幕 | 布局变化 |
|------|----------|
| 桌面端 (>992px) | 左右两栏并排 |
| 平板端 (≤992px) | 作者卡片移至顶部，水平排列 |
| 手机端 (≤768px) | 单列布局，卡片图片在上 |

### 深色模式

- 卡片背景：`#2a2a2a`
- 文字颜色：自动调整为浅色
- 阴影效果：使用 Miku 绿色光晕

## 使用指南

### 添加新作品

复制以下代码块到 `.article-list` 内：

```html
<div class="article-card" data-id="新ID">
    <div class="article-info">
        <span class="article-category">分类</span>
        <h2 class="article-title">标题</h2>
        <p class="article-excerpt">简介...</p>
        <span class="article-date">日期</span>
    </div>
    <div class="article-cover">
        <img src="图片路径" alt="描述">
    </div>
</div>
```

### 修改作者信息

编辑 `.author-card` 内的对应元素即可。

### 自定义社交链接图标

可以使用 emoji 或 Font Awesome 图标：

```html
<a href="https://pixiv.net/..." title="Pixiv">🎨</a>
<a href="https://twitter.com/..." title="Twitter">🐦</a>
```
