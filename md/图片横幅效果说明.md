## 🎯 关键CSS效果

### 1. 图片虚化效果
```css
.hero-image {
    filter: blur(8px) brightness(0.7);
    transform: scale(1.1);
}
```
- `blur(8px)` - 8像素的高斯模糊
- `brightness(0.7)` - 降低亮度到70%，让文字更突出
- `scale(1.1)` - 放大1.1倍，避免模糊后边缘出现白边

### 2. 半透明遮罩
```css
.hero-overlay {
    background: linear-gradient(
        to bottom,
        rgba(0, 0, 0, 0.3) 0%,
        rgba(0, 0, 0, 0.5) 100%
    );
}
```
- 从上到下的渐变遮罩
- 上方30%透明度，下方50%透明度
- 让文字更清晰可读

### 3. 标题居中
```css
.hero-content {
    position: relative;
    z-index: 2;
    text-align: center;
}
```
- 使用相对定位
- z-index确保在图片和遮罩之上
- 文字居中对齐

### 4. 文字阴影
```css
.hero-title {
    text-shadow: 0 4px 15px rgba(0, 0, 0, 0.5);
}

.hero-subtitle {
    text-shadow: 0 2px 10px rgba(0, 0, 0, 0.5);
}
```
- 添加阴影让文字在亮色背景上也清晰可见
- 增强文字的立体感

## 🎨 样式定制

### 修改虚化程度
```css
.hero-image {
    filter: blur(5px) brightness(0.7);  /* 减少模糊 */
    /* 或 */
    filter: blur(12px) brightness(0.7); /* 增加模糊 */
}
```

### 修改横幅高度
```css
.hero-banner {
    height: 600px;  /* 默认500px */
}
```

### 修改遮罩深度
```css
.hero-overlay {
    background: linear-gradient(
        to bottom,
        rgba(0, 0, 0, 0.5) 0%,   /* 改成0.5，更深 */
        rgba(0, 0, 0, 0.7) 100%  /* 改成0.7，更深 */
    );
}
```

### 更换图片
```html
<img src="../images/picture/你的图片.jpg" alt="Miku" class="hero-image">
```

### 修改标题大小
```css
.hero-title {
    font-size: 5rem;  /* 默认4rem */
}

.hero-subtitle {
    font-size: 1.8rem;  /* 默认1.5rem */
}
```

## 📱 响应式设计

### 桌面端（>768px）
- 横幅高度：500px
- 主标题：4rem
- 副标题：1.5rem
- 内容区域：最大宽度900px居中

### 移动端（≤768px）
- 横幅高度：400px
- 主标题：2.5rem
- 副标题：1.1rem
- 内容区域：全宽，减少内边距

## 💡 设计要点

### 1. 虚化效果的作用
- 减弱背景图片的干扰
- 突出前景的文字内容
- 营造朦胧美感
- 保持图片的氛围感

### 2. 遮罩层的作用
- 增强文字对比度
- 让白色文字更清晰
- 渐变效果更自然
- 统一视觉风格

### 3. 标题阴影的作用
- 在亮色背景上保持可读性
- 增加立体感
- 提升视觉层次
- 强化焦点

## 🔧 常见调整

### 1. 图片太亮，文字看不清
```css
.hero-image {
    filter: blur(8px) brightness(0.5);  /* 降低亮度到50% */
}

/* 或者加深遮罩 */
.hero-overlay {
    background: rgba(0, 0, 0, 0.6);  /* 单色深遮罩 */
}
```

### 2. 虚化不够柔和
```css
.hero-image {
    filter: blur(15px) brightness(0.7);  /* 增加模糊 */
}
```

### 3. 标题太大或太小
```css
.hero-title {
    font-size: 3rem;  /* 调整大小 */
}
```

### 4. 想要清晰的图片背景
```css
.hero-image {
    filter: none;  /* 移除所有滤镜 */
    /* 或只保留亮度调整 */
    filter: brightness(0.7);
}
```
